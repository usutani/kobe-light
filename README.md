# Kobe Light — Omarchy Theme

Omarchy カスタムテーマ。

## 適用

```bash
omarchy theme set kobe-light
```

## 構成

- **Neovim** — standalone 実装の colorscheme
- **Waybar** — ステータスバー色
- **Hyprland** — アクティブウィンドウ枠色
- **Hyprlock** — ロック画面色
- **Mako** — 通知デーモン色
- **Walker** — アプリランチャー色
- **SwayOSD** — OSD 表示色
- **Obsidian** — ノートアプリテーマ
- **btop** — リソースモニター色
- **Chromium** — 新規タブ背景色
- **壁紙** — 同梱（`backgrounds/`）
  - `suma-coast.jpg` — 須磨海岸（提供：神戸市 / CC BY-NC-SA 4.0, [PHOTO PORT](https://www.photoport-kobe.jp/photo/836)）
